Informasjon

Kilde til data for oppgaven ligger i en SQLite databasen som heter data.db. Filen kan lastes ned her.

Databasen inneholder 3 tabeller:

  • data2020 : Data for kommuner i 2020
  • data2021 : Data for kommuner i 2021
  • tblCode : Kodebok for omkoding av kjønn

Disse kolonner finnes i dataene for kommuner i tabell data2020:

  • GEO : Geografiske koder for kommuner. Manglende kode er kodet med 9999
  • AAR : Årgang for data innsamling
  • ALDER : Alder for personer
  • KJONN : Kjønn for personer
  • ANTALL : Antall personer med samme ALDER og KJONN per GEO og AAR

Disse kolonner finnes i dataene for kommuner i tabell data2021:

  • ALDER : Alder for personer
  • KJONN : Kjønn for personer
  • X… : Kolonnenavn starter med X etterfulgt med 4 siffer representerer geografiske koder. Kolonnen inneholder antall personer per ALDER og KJONN.

Tabell for å koble geografiske koder finnes hos SSB og kan lastes ned som en csv fil fra denne URL-en:

https://www.ssb.no/klass/klassifikasjoner/127/versjon/1159/korrespondanser/454.

Hvis du bruker R, kan norgeo pakken brukes for å hente data direkte istedenfor å laste ned csv filen:

install.packages("norgeo")
library(norgeo)
geoKode <- get_correspond("fylke", "kommune")

Hvis du vil lese hvordan get_correspond() fungerer så finnes beskrivelsen her:

https://rdrr.io/cran/norgeo/man/get_correspond.html

Oppgaven

Bruk analyseverktøy du kjenner for å løse disse oppgavene:

  1. Hvor mange menn og kvinner er det i 2021-datasettet og i hele datasettet?
  2. Hvor mange fylker finnes i datasettet?
  3. Hvor mange personer er 19 år i Agder fylke (kode 42)?

Tips

  • Fylkekoder (2 siffer) fra SSB må kobles til ved bruk av kommunekoder (4 siffer) som koblingsid.
