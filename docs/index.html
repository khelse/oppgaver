Oppgave

Kilde til data for oppgaven ligger i en SQLite databasen som heter oppgave.db. Databasen inneholder 3 tabeller:

  • data2020 : Data for kommuner i 2020
  • data2021 : Data for kommuner i 2021
  • tblCode : Kodebok

Disse kolonner finnes i dataene for kommuner:

  • GEO : Geografisk koder for kommuner. Manglende kode er kodet som 9999
  • AAR : Årgang for data innsamling
  • ALDER : Alder for personer
  • KJONN : Kjønn for personer
  • ANTALL : Antall personer med samme ALDER og KJONN per GEO og AAR

Tabell for å koble geografiske koder finnes hos SSB som kan lastes ned som csv fil her https://www.ssb.no/klass/klassifikasjoner/127/versjon/1159/korrespondanser/454.

Hvis du bruker R, kan norgeo pakken brukes for å hente kodene direkte:

install.packages("norgeo")
library(norgeo)
geoKode <- get_correspond("fylke", "kommune")

URL-en nedenfor beskriver hvordan get_correspond() fungerer https://helseprofil.github.io/norgeo/reference/get_correspond.html

Bruk analyse verktøy du kan best for å løse disse oppgavene:

  1. Hvor mange menn og kvinner som finnes i datasettet?
  2. Hvor mange fylker finnes i datasettet? Tips: Fylkekoder fra SSB må kobles til ved bruk av kommunekoder som koblingsid.
  3. Hvor mange personer som er 19 år i Agder fylke (kode 42)?
